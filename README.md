割り勘計算機システム<br>
請求額と、グループでの分割もできる。<br>
例えば、男女別グループで分割したい場合は、男性の人数女性の人数をそれぞれ入力すると、男性一人分女性一人分の金額が出る。<br>
また、支払い倍率は二グループでの倍率である<br>
さらに支払い単位数を決めることができ、端数分も出力される。<br>
例えば、単位を千円として、3300円支払う際は、一人当たり千円で端数は300である。
